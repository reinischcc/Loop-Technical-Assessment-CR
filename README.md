# Automation Test - Playwright Assessment

## Setup & Run

```bash
npm install
npm test
```

That's it. `npm test` installs the required browser and runs all 6 tests automatically.

## Project Structure
├── tests/
│   ├── tasks.spec.ts      # All 6 tests generated from JSON
│   └── testData.json      # Credentials and test case data
├── playwright.config.ts
├── tsconfig.json
└── package.json
